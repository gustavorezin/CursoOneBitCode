## INICIAR PROJETO

- npm init -y

## INSTALAR TS

globalmente

- npm i -g typescript

workspace

- npm i --save-dev typescript

## COMPILAR O PROJETO

- tsc nomearquivo.TS
- ou configurar tsconfig.json => AUTOMATICAMENTE COM tsc --init

## TESTAR O PROJETO

- node nomearquivo.JS
- caso tsconfig.json configurado => node main.JS
- caso package.json configurado => npm run watch
