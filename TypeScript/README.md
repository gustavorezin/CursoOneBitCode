## INICIAR PROJETO

- npm init

## INSTALAR TS

globalmente

- npm i -g typescript

workspace

- npm i --save-dev typescript

## COMPILAR O PROJETO

- tsc nomearquivo.TS
- ou configurar tsconfig.js => AUTOMATICAMENTE COM tsc --init

## TESTAR O PROJETO

- node nomearquivo.JS
- caso tsconfig.js configurado => node main.JS
